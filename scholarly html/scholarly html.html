1. Abstract

Zoo Web Manager is a web application intended to manage information related to a zoo garden.

2. Introduction

The purpose of this web application is to facilitate the management of animals in a zoo. Through it, users can search for animals using different filters provided by the developers.

3. Structure

The architecture of this web application is based on multiple pages that are easy to navigate and use by a regular user: My Account, Animals, Log In, Sign Up, Buy Ticket, About.

3.1 About (pagina_about.html)

This is the main page of the application. After the user logs in, this is the first page that introduces him to our website.

Author: Cernat George Lucian

3.2 Log In (index.html)

A page where the users can log in their accounts(they can be visitors or zoo staff(not implemented)).

Author: Popa Ioana

3.3 Sign In (SignIn.html)

This is the page a visitor can create a user account, or a zoo staf account.

Author: Popa Ioana

3.4 My account(pagina_my_account.html)

This is the page where the user can check the details about his account, change his password or log out.

Author: Cernat George Lucian

3.5 Change Password (pagina_change_password.html)

This is the page where a user that is logged in can change his password.

Author: Cernat George Lucian

3.6 Animals (pagina_animals.html)

This is the page where a user can search and discover different species of animals, searching them using the filters.

Author: Cernat George Lucian

3.6.1 Animal (animale.html)

A page that gives specific information about a certain animal (in our case the lion).

Author: Bors Andrei Darius

3.7.1. Reptile House (casa_reptilelor.html)

A page for the zoo's house of reptiles, builded to give the user the sensation that he/she is an underground reptilian den.

Author: Bors Andrei Darius

3.7.2. Halls Of the Depths (hallofthedeapths.html)

A page for the zoo's aquarium, builded to give the sensation for he user that he/she is in the sea.

Author: Bors Andrei Darius

3.7.3. Playground (PlayGround.html)

A page made to inform the users about the zoo's playground, as well as the animals that the kids may play with.

Author: Bors Andrei Darius

3.8 Buy Ticket (pagina_day_ticket.html,pagina_monthly_pass.html)

This pages are designed to help the user in buying tickets easily online.

Author: Cernat George Lucian

4. Navigate

As a user you can easily log in, or create an account if you don't have one. After you log in you are welcomed by the about page which is the main page of the web application. Using the top navigation bar you can navigate through different pages and functionalities, like discovering the uknown things about animals by using the filter, buy tickets, see the main attractions and manage your account profile.
